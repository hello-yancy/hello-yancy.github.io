Vue.js（读音 /vjuː/, 类似于 view） 是一套构建用户界面的渐进式框架。
Vue 只关注视图层， 采用自底向上增量开发的设计。
Vue 的目标是通过尽可能简单的 API 实现响应的数据绑定和组合的视图组件。

# 安装
Vue.js提供一个命令行工具vue-cli，可用于快速搭建大型单页应用
```
$npm install vue-cli -g

```
